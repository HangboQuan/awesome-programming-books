# cs-books
计算机学科日新月异，今天不学习，明天变垃圾。为了节省大家搜寻电子书的时间，为此我自己整理了比较经典的计算机专业书籍，供大家阅览，涉及计算机基础、数据结构和算法、Java/Scala/C++/Python/Go/JavaScript/Vue、网络、操作系统、Linux、程序员数学、程序员英语等电子书。愿大家升值加薪，走上人生巅峰。<br>
